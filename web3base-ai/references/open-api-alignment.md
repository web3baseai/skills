# Open API alignment

This unified skill uses Web3Base public Open API endpoints:

- `GET /api/open/chains`
- `GET /api/open/twitter/users`
- `GET /api/open/youtube/channels`
- `GET /api/open/nav/groups`
- `GET /api/open/nav/groups/:slug`
- `GET /api/open/nav/menus/:pathSlug`
- `GET /api/open/category-groups?module=chain|twitter|youtube`
- `GET /api/open/categories?scope=chain|twitter|youtube|item`
- `GET /api/open/spec`
